# 👥 KeenKeeper

## Description
KeenKeeper is a friendship management web application that helps users keep track of their friends, monitor communication frequency, and maintain stronger relationships through calls, texts, and video interactions.

## Technologies Used
- React.js
- React Router DOM
- Tailwind CSS
- DaisyUI
- Recharts
- React Toastify

## Key Features
- View friend profiles and track contact status.
- Log Call, Text, and Video interactions with timeline history.
- Visualize communication activity using analytics charts.
