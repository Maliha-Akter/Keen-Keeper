# 👥 KeenKeeper

## Description
KeenKeeper is a friendship management web application that helps users keep track of their friends, monitor communication frequency, and maintain stronger relationships through calls, texts, and video interactions.

## Technologies Used
- React.js
- React Router DOM
- Tailwind CSS
- DaisyUI
- Recharts
- React Toastify

## Key Features
- Friend profiles and contact statuses are displayed and monitored.
- Call, Text, and Video interactions are recorded and shown in a timeline history.
- Communication activity is visualized through analytics charts.
